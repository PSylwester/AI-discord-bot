## How to implement repositorium in github desktop.

1. Download `github desktop`,
2. connect your account on github with application,
3. copy url repositorium by clicking `code`,
4. then `copy` repo url,
5. create folder for example `Project` wherever you want,
6. in application choose `file` -> `Clone repository` -> `URL`,
7. paste url repositorium in github desktop and choose your created folder `Project` -> click `Clone`,
8. now you can open VS Code on your Github dashboard and you should have your repository connected properly.
